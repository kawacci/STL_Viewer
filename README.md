# M5Tab5 STL/OBJ Viewer Ver.3.0(Auto-Rotation & 3D Control)

M5Tab5でSTLファイルをリアルタイム表示・操作するための3Dビューワーです。
IMU（加速度センサ）を利用し、デバイス90度毎に画面と操作系が自動的に回転する。

## ✨ 主な機能
- **STL / OBJ 両対応**: STLファイルだけでなく、OBJファイルも読み込み可能。
- **起動時ランチャー**: 起動後にSDカード内の`/STL_OBJ`フォルダからモデルを選択できる。
- **リアルタイム3Dレンダリング**: microSDカード内のモデルファイルを読み込み描画。
- **IMU連動自動回転**: 本体の向き90度毎に、画面表示・タッチ座標・3D回転軸がすべて自動で追従。
- **インテリジェント・タッチ操作**:
  - **1本指スワイプ**: モデルの回転（画面の向きに関わらず、指の動きにモデルが吸い付くように補正）。
  - **2本指ピンチ**: ズームイン/ズームアウト。
- **ダイナミックUI**:
  - モデル名とポリゴン数をリアルタイム表示。
  - カラーパレットによるモデル色の即時変更。
  - 一定時間無操作で自動回転展示（AUTOモード）に移行。

## 🛠 ハードウェア要件
- **本体**: [M5Stack Tab5](https://docs.m5stack.com/ja/core/M5Tab5)
- **ストレージ**: microSDカード
- **配置フォルダ**: SDカード内の `/STL_OBJ` フォルダにSTL/OBJファイルを入れてください。

## 🚀 セットアップと使い方
1. **ライブラリの準備**: Arduino IDE または PlatformIO で以下のライブラリを導入してください。
   - `M5Unified`
   - `M5GFX`
2. **ファイル配置**: 表示したいモデルをSDカードの `/STL_OBJ` フォルダにコピーします。
   - **ポリゴン数**: 3Dデータは1万ポリゴン程度に抑えてください（パフォーマンスのため）。
   - **OBJファイルの場合**: MTLファイルもOBJファイルと一緒に `/STL_OBJ` フォルダに保存してください。
3. **書き込み**: M5Tab5 を接続してビルド・書き込みを行ってください。


-  **Note**: コンパイルには M5Stack Board Manager v3.2.0 以上を使用し、ボード選択で M5Tab5 を指定してください 。

## 💡 技術的な工夫
-  **鏡像修正の克服**: STLデータの特性（鏡像）とディスプレイの座標系を合わせるため、読み込み時にX軸を反転。それに伴うポリゴンの表裏判定（バックフェース・カリング）の逆転を数学的に解決しました。

-  **座標正規化**: 画面回転時でも、物理的なタッチパネル座標を論理座標に変換することで、UIの当たり判定と3D操作の一貫性を保っています。

-  **高速描画**: メモリ消費を抑えるため、スプライトを用いたタイル分割レンダリングを採用しています。
